📊 HR Analytics Dashboard — Microsoft Excel
An interactive HR Analytics Report built entirely in Microsoft Excel, 
providing a comprehensive overview of workforce data including salary trends,
employee demographics, job satisfaction, and work-life balance metrics.

🛠️ Tools & Features Used
Microsoft Excel

Pivot Tables — for summarising employee data by gender, education, employment type, and marital status
Pivot Charts — bar charts, line charts, and pie charts for visual data representation
Slicers — interactive filter buttons for Gender, Education, EmploymentType, MaritalStatus, JobSatisfaction, WorkLifeBalance, and Month
Power Query — for data cleaning and transformation
Power Pivot / Data Model — for handling relationships between multiple data tables
DAX Measures — for calculated KPIs such as Total Salary Paid and Current Employee Headcount
Conditional Formatting — for visual highlights across KPI cards
Named Ranges & Dynamic References — to keep charts and summaries responsive to filters
